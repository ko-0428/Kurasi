# くらしリフォーム｜ホームページ

リフォーム会社「くらしリフォーム」のコーポレートサイト（1ページ完結型）です。
Claude Design のデザイン（`くらしリフォーム.dc.html`）をもとに、実際に公開できる静的サイトとして実装しました。

## ファイル構成

```
.
├── index.html        … ページ本体（HTML）
├── styles.css        … デザイン・レイアウト（配色 / レスポンシブ / ホバー）
├── images/           … 公開用の写真（Webに最適化済み JPEG）
│   ├── hero.jpg              … ヒーロー（メインビジュアル）
│   ├── service-kitchen.jpg   … サービス：キッチン
│   ├── service-bath.jpg      … サービス：浴室・洗面
│   ├── service-interior.jpg  … サービス：内装
│   ├── service-exterior.jpg  … サービス：外壁・外構
│   ├── work-kitchen.jpg      … 施工事例：ナチュラルキッチン改装
│   ├── work-bath.jpg         … 施工事例：明るい浴室リフォーム
│   ├── work-renovation.jpg   … 施工事例：中古住宅リノベーション
│   ├── work-exterior.jpg     … 施工事例：外壁リフレッシュ
│   ├── about.jpg             … 私たちについて（打ち合わせ風景）
│   ├── president.jpg         … ごあいさつ（代表者 渡邊陽介 様の写真）
│   ├── cta.jpg               … お問い合わせ（バナー写真）
│   └── source/              … 差し替え前のオリジナル画像（公開には未使用・バックアップ）
└── README.md
```

## 写真の差し替え方

`images/` フォルダ内の **同じファイル名** で写真を上書きするだけで反映されます（HTML の編集は不要）。

- 例：トップの大きな写真を変えたい → 新しい写真を `hero.jpg` という名前にして `images/` に上書き保存
- 形式は JPEG 推奨。横幅の目安は ヒーロー=1600px / その他=640px 程度、「私たちについて」=1100px 程度
- 表示位置は自動でトリミング（中央基準で `object-fit: cover`）されます

オリジナルの高解像度画像は `images/source/` にバックアップしてあります。

## テーマカラーの変更

`styles.css` 先頭の `:root` でサイト全体の色を一括変更できます。

```css
:root {
  --accent: #E8913C;       /* メインカラー（オレンジ） */
  --accent-dark: #D67E26;  /* ホバー時などの濃い色 */
}
```

元デザインには他のテーマ候補もありました。差し替える場合は上記2色を置き換えてください。

| テーマ | --accent | --accent-dark |
|--------|----------|----------------|
| オレンジ（現在） | `#E8913C` | `#D67E26` |
| セージ（緑） | `#7E9A6E` | `#5F7B50` |
| ネイビー（紺） | `#4A7BA6` | `#345C82` |

## テキスト・連絡先の変更

電話番号・住所・会社名などは `index.html` 内に直接記載しています。

- 電話番号：`0120-123-456`（`tel:` リンクと表示の2か所、フッターにも記載）
- 住所・会社情報：フッター（`footer-brand`）
- メニュー項目・キャッチコピー・サービス内容・お客様の声：各セクションのテキストを編集
- 社長挨拶（`MESSAGE` セクション）：代表者名は「渡邊陽介」、写真は `images/president.jpg`（ご提供いただいた写真）を設定済みです。デザインは、波形にカットした写真＋筆記体の署名（`Yosuke Watanabe`／Google Fonts の Allura）＋重ねた白いカード（ごあいさつ・氏名）＋下部のベージュのあいさつ文パネル、という構成です。写真を変更する場合は同じファイル名 `images/president.jpg` で上書きしてください（顔の位置は `styles.css` の `.message-photo`／`<image>` の `preserveAspectRatio` 付近で調整、波の形は同ファイル内 `clipPath#presWave` のパスで調整できます）。あいさつ本文は仮テキストのため、`index.html` の `message` セクションで実際の内容に差し替えてください。

## 公開（デプロイ）

静的サイトなのでビルド不要です。`index.html`・`styles.css`・`images/`（`source/` を除く）をそのままホスティングに上げれば公開できます。

- Vercel の場合：このフォルダで `vercel` を実行（`.vercelignore` でバックアップ画像は除外済み）
- そのほか Netlify、レンタルサーバー（FTP アップロード）などでも同様に公開できます

## メモ

- 元の Claude Design データの画像は API の取得上限（256KiB）により完全には取得できなかったため、ご用意いただいた写真を最適化して使用しています。
- レイアウトはPC・タブレット・スマートフォンに対応（レスポンシブ）。スマホではメニューがハンバーガー（≡）に切り替わります。
